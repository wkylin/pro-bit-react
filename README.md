# Bit.dev

## 安装

1. npm i -g @teambit/bvm
2. bvm install
3. bvm upgrade

## 初化Workspace

1. dir pro-bit
2. cd pro-bit && bit init

## 模板 & 创建组件
1. bit template
2. bit new react pro-ui/pro-react/sale-charge --default-scope wkylin.wui

## 安装依赖
1. bit install -u @testing-library/react@12.1.5 

## Tag
1. bit tag --all

## 部署
1. bit export
